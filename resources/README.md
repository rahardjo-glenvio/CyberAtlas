# Resources

Curated references for going deeper.

This section collects books, courses, tools, and references that are worth your time. It is not a comprehensive list of everything available. It is a selective list of resources that offer genuine value — with brief notes explaining what each one is good for.

---

## Selection Criteria

A resource is included here if it meets at least one of these criteria:

- It is widely recognized as a foundational or authoritative source
- It explains a complex topic better than alternatives
- It is free and high-quality
- It fills a gap not well covered elsewhere

A resource is not included just because it exists or is popular. Quantity is not the goal.
