# Cryptography

Encryption, hashing, public key infrastructure, and how cryptography fails in practice.

Cryptography underpins almost every security mechanism: encrypted communications, password storage, certificate trust, digital signatures. Understanding it does not require advanced mathematics, but it does require understanding what guarantees cryptographic primitives provide and where those guarantees break down.
