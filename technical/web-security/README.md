# Web Security

Web application vulnerabilities, HTTP mechanics, and client/server security.

Web applications are one of the largest and most accessible attack surfaces in modern organizations. They expose functionality over HTTP to the internet, often with complex business logic that creates security gaps independent of underlying platform security.
