# Incident Response

The process of detecting, containing, eradicating, and recovering from security incidents.

Incident response is where security knowledge gets tested under pressure. The IR lifecycle is well-defined, but executing it well requires preparation, clear processes, and the ability to investigate quickly without destroying evidence.
