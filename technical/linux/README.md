# Linux

Linux internals, permissions, and security fundamentals.

Linux is the dominant operating system in server environments, cloud infrastructure, and security tooling. Understanding it from a security perspective means knowing how the permission model works, where sensitive data lives, and what attackers look for after gaining access.
