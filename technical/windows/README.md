# Windows

Windows internals, authentication, and security architecture.

Windows dominates enterprise desktop environments and is the target of the majority of advanced threats. Understanding Windows security requires understanding its authentication stack, its registry, its event logging system, and the many ways its defaults favor compatibility over security.
