# Cloud Security

Cloud infrastructure security — IAM, misconfigurations, and the shared responsibility model.

Cloud environments introduce a fundamentally different security model. The attack surface shifts from network perimeter to identity and configuration. Most cloud breaches trace back not to sophisticated exploits, but to misconfigured permissions and exposed credentials.
