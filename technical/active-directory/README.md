# Active Directory

Active Directory architecture, authentication, and attack paths.

Active Directory is the identity backbone of most enterprise Windows environments. Understanding AD is essential for both attackers — who exploit it to move laterally and escalate privileges — and defenders who must secure an environment built on it.
